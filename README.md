## Getting Started
- Go Version : 
  - `<=1.24.0`

- IDE : 
  - `VSCode`
  - `Trae`

- DB : 
  - `MySQL`

```bash
# Path : Go-Server/.
go build
```

## .ENV-Configure
```env
MYSQL_USERNAME=YOUR_DB_USERNAME
MYSQL_USERPASSWORD=YOUR_DB_PASSWORD
MYSQL_HOSTNAME=YOUR_DB_HOSTNAME
MYSQL_PORT=YOUR_DB_PORT
```
